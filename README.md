# DeltaDesk — Telegram-бот цен с бирж

Бот: **https://t.me/DeltaDesk1_bot**

Сравнивает цены на **Binance, Bybit, OKX, Kraken, KuCoin, Gate, MEXC** и показывает, где выгоднее покупать и продавать. Есть упрощённый анализ тренда.

| Режим | Инструкция |
|-------|------------|
| **24/7 бесплатно (облако)** | [DEPLOY_FREE.md](DEPLOY_FREE.md) |
| **На своём ПК** | `setup.bat` → `.env` → `start.bat` |

**Python ставить вручную не обязательно** — `setup.bat` попробует установить его сам (Windows 10/11).

---

## Быстрый старт (3 шага)

### 1. Создайте бота в Telegram

1. Откройте [@BotFather](https://t.me/BotFather)
2. Отправьте `/newbot`
3. Придумайте имя и username
4. Скопируйте **токен** (длинная строка вида `123456789:ABC...`)

### 2. Установка на компьютере

1. Откройте папку `crypto-telegram-bot`
2. Дважды щёлкните **`setup.bat`**
3. Если Python не был установлен — после установки **запустите `setup.bat` ещё раз**
4. Откройте файл **`.env`** блокнотом и вставьте токен:

```
BOT_TOKEN=ваш_токен_от_BotFather
```

### 3. Запуск

Дважды щёлкните **`start.bat`**. Окно должно оставаться открытым — пока оно открыто, бот отвечает в Telegram.

В Telegram напишите боту:

- `/price BTC`
- `/analyze ETH`
- или просто `SOL`

---

## Если автоматическая установка Python не сработала

1. Скачайте Python: https://www.python.org/downloads/
2. При установке **обязательно** включите галочку **«Add python.exe to PATH»**
3. Снова запустите `setup.bat`

## Команды бота

| Команда | Описание |
|---------|----------|
| `/price BTC` | Цены на ваших биржах |
| `/analyze BTC` | Изменение за ~24ч и ~7д, тренд SMA, волатильность |
| `BTC` | То же, что `/price BTC` |
| `/exchanges` | Какие биржи у вас включены |
| `/add bitget` | Добавить биржу в свой список |
| `/remove mexc` | Убрать биржу |
| `/reset` | Вернуть стандартный список |

Настройки **сохраняются отдельно для каждого пользователя** Telegram (файл `user_data.json`).

## Важно

- Бот использует **публичные** данные бирж, API-ключи не нужны.
- Учитываются не комиссии, не вывод, не проскальзывание — только ориентир по цене.
- Анализ — упрощённый технический обзор, **не** инвестиционная рекомендация.
- Для работы **24/7** см. [DEPLOY_FREE.md](DEPLOY_FREE.md) (Fly.io). На ПК бот работает только пока открыт `start.bat`.
- **Не публикуйте BOT_TOKEN** — при утечке смените через @BotFather → Revoke.
